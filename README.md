## Usage

1. Read the book


2. Make `git checkout` to tag according your chapter

    ```bash
    $ git checkout 'Chapter_18_First_Steps_to_xUnit'
    ```


3. View report according to your current branch [REPORT](REPORT.md)


4. Run unit test
    ```bash
    $ php xunit.php
    ```
